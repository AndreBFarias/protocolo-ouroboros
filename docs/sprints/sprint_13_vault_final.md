# Sprint 13 -- Integração Vault Final

## Status: Pendente
Issue: a criar

## Objetivo

Centralizar todo o ecossistema pessoal em um único vault. Avaliar melhor estratégia de unificação e integrar documentos de diferentes domínios.

## Entregas

- [ ] Avaliar melhor estratégia (mover projeto pro vault ou vault pro projeto)
- [ ] Integrar documentos pessoais (contratos, currículos, diplomas)
- [ ] Integrar documentos acadêmicos
- [ ] Integrar documentos profissionais (MEC/G4F, Energisa)
- [ ] Criar módulos novos (tracker acadêmico, carreira, saúde)
- [ ] Integração com celular (Obsidian mobile sync)
- [ ] Unificar como "novo Controle de Bordo"

## Armadilhas conhecidas

- Mover o projeto pode quebrar paths relativos em scripts e imports
- Vault grande demora para sincronizar no mobile
- Obsidian Sync pago vs alternativas gratuitas (Syncthing, Git)
- Documentos sensíveis (contratos, diplomas) precisam de cuidado extra no .gitignore
